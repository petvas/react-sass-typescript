# react-sass-typescript

Minimum Boilerplate with React Sass and TypeScript

## Contains
- [x] [Typescript](https://www.typescriptlang.org/) 3
- [x] [React](https://facebook.github.io/react/) 16.4
- [x] [Sass](https://sass-lang.com/)

## Setup
```
$ npm install
```

## Running
```
$ npm run start
```

## Build
```
$ npm run build
```
